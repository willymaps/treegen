# Mapbox Tree Generator

A working example of generating 3D trees using [Turf.js](http://turfjs.org/) and Mapbox GL JS [fill-extrusions](https://docs.mapbox.com/mapbox-gl-js/style-spec/layers/#fill-extrusion).
